# Инструкция по установке Python и запуску PaintPro

## Проблема
У вас установлен только алиас Python из Windows Store, а не полноценный Python.

## Решение

### Вариант 1: Установка через Microsoft Store (Рекомендуется)
1. Откройте Microsoft Store
2. Найдите "Python 3.12" или "Python 3.11"
3. Нажмите "Установить"
4. После установки перезапустите PowerShell
